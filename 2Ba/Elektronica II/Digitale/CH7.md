### The D Flip-Flop

* D hoog
    * Q hoog op trigger van de CLK
    * SET
* D laag
    * Q laag op trigger van de CLK
    * RESET
* Q volgt D op de trigger edge van de CLK

    * Flip-Flop SET

    * Q laag op trigger van de CLK
    * Flip-Flop RESET
